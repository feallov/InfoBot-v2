# Ultimate Telegram Bot 🤖

Многофункциональный Telegram-бот с админ-панелью, AI-чатом, погодой, напоминалками, генератором паролей и троллинг-функциями.

## Возможности

- 🌤 **Погода** — актуальный прогноз по городу (OpenWeatherMap)
- 🤖 **AI Чат** — диалог с нейросетью через OpenRouter (бесплатные модели)
- ⏰ **Напоминалки** — установка таймеров на нужное время
- 🔐 **Генератор паролей** — безопасные пароли разной длины
- 👤 **Профиль** — информация о пользователе
- 🔧 **Админ-панель** — рассылки, бан/разбан, статистика, управление API ключами
- 😈 **Троллинг-меню** — фейковые системные сообщения, "взлом", лотерея, glitch-текст, zalgo и многое другое

## Локальный запуск

```bash

source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Деплой на Render (Web Service / Background Worker)

Рекомендуется использовать **Background Worker** для ботов на polling.

### Через Background Worker (polling)
1. Запушьте проект на GitHub.
2. В [Render Dashboard](https://dashboard.render.com) создайте **Background Worker**.
3. Укажите:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python main.py`
4. Добавьте переменные окружения (`Environment`) в настройках сервиса.
5. Нажмите **Deploy**.

### Через Web Service (webhook) — если хотите, чтобы бот не засыпал
1. Создайте **Web Service**.
2. Укажите **Start Command**: `python main.py`.
3. Добавьте переменную окружения:
   ```
   WEBHOOK_URL=https://your-service-name.onrender.com/webhook
   ```
   (Render выдаст URL после создания сервиса, пересоздайте переменную и перезапустите, если URL ещё не известен).
4. Бот поднимет веб-сервер и установит webhook автоматически.

## Панель администратора

В личном чате с ботом отправьте `/admin` (только для `ADMIN_ID`).

Доступные функции админки:
- 📊 Статистика по пользователям и сообщениям
- 📢 Рассылка всем пользователям
- 🚫 Бан / Разбан
- 🔑 Установка/смена AI API ключа прямо из бота
- 😈 Троллинг-меню с фейковыми сообщениями и text-mutations

## Троллинг-функции

Все отправляются от имени бота конкретному пользователю по его TG ID:
- **Zalgo / Glitch / Reverse / Mocking** — искажение текста
- **Фейк системное уведомление** — выглядит как системное сообщение
- **Фейк взлом** — сообщение о "взломе аккаунта"
- **Фейк лотерея** — поздравление с выигрышем
- **Фейк бан** — сообщение о блокировке
- **Фейк погода** — сгенерированный прогноз с любыми данными

> ⚠️ Используйте троллинг только для друзей и с их согласия. Будьте осторожны с массовыми рассылками, чтобы не получить ограничения от Telegram.

## База данных

По умолчанию используется **SQLite** (`bot.db`). Для Render на бесплатном тарифе файлы в Background Worker сохраняются между рестартами, но если хотите гарантированную персистентность, подключите **PostgreSQL** (Render предоставляет бесплатный инстанс) и замените `db.py` на `asyncpg` или `psycopg`.
